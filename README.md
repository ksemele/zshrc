# How to

Clone this repo to `$HOME`.
Create `$HOME/.private.zshrc` with keys and tokens.
Add extra files to `.zshrc`:
```
cat ~/.zshrc
source $HOME/zshrc/public.zshrc
source $HOME/.private.zshrc

# other specific envs
source $HOME/.project.zshrc
```
